## アプローチ1: ネオ・ピクセルパンク

<response>
<text>
**Design Movement**: ネオ・ピクセルアート × サイバーパンク
**Core Principles**:
- 8bitピクセルアートの美学を現代的なグロー効果と融合
- 暗い背景に蛍光色のアクセントで視覚的コントラスト
- ゲームUIのHUD要素（ライフゲージ、スコア表示）をナビゲーションに転用
- ステージ構造はロックマン6ステージ構成を踏襲

**Color Philosophy**: 深夜の都市をイメージ。背景は#0a0a1a（深い紺黒）、プライマリは#00ffcc（サイバーティール）、アクセントは#ff3366（ネオンマゼンタ）。敵キャラクターは赤系、アイテムは黄色。

**Layout Paradigm**: 横スクロール固定視点。画面幅いっぱいのキャンバス。上部にHUD（HP・スコア・ステージ名）、下部に地面プラットフォーム、中央にゲームフィールド。

**Signature Elements**:

- ピクセルフォント（Press Start 2P）でのテキスト表示
- スキャンライン効果のオーバーレイ
- グロー（発光）エフェクトのあるプラットフォームと敵

**Interaction Philosophy**: キーボード操作（矢印キー/WASD）でキャラクター移動。情報ギミックに触れると画面がフラッシュしてモーダル表示。

**Animation**: キャラクターはスプライトアニメーション（2フレーム交互）。敵は上下にホバリング。背景は視差スクロール（パララックス）。

**Typography System**: Press Start 2P（ゲームUI・見出し）+ Rajdhani（本文・説明）
</text>

## アプローチ2: ダーク・インダストリアル・ピクセル

<response>
<text>
**Design Movement**: インダストリアル・ハードコア × クラシックアーケード
**Core Principles**:
- 工場・廃墟をテーマにしたステージ背景
- モノクロベースに血赤のアクセント
- ノイズテクスチャとグランジ感
- ハードロック/メタルバンドの荒々しさを視覚化

**Color Philosophy**: 錆びた金属と血の赤。背景は#111111（ほぼ黒）、構造物は#333333（ダークグレー）、アクセントは#cc0000（ディープレッド）、ハイライトは#ff6600（バーンオレンジ）。

**Layout Paradigm**: 縦に複数レイヤーのプラットフォーム。地下から地上へ上昇するステージ構造。

**Signature Elements**:

- 錆びたメタルテクスチャのプラットフォーム
- 火花エフェクトと煙
- ドラム缶・パイプなどの障害物

**Interaction Philosophy**: 敵を倒すと情報が解放される。ボス戦形式でセクションにアクセス。

**Animation**: 重厚感のある動き。ジャンプ時の衝撃波。敵撃破時の爆発エフェクト。

**Typography System**: Metal Mania（タイトル）+ Source Code Pro（本文）
</text>

## アプローチ3: レトロ・ネオン・アーケード（採用）

<response>
<text>
**Design Movement**: 80年代アーケードゲーム × ロックマン直系オマージュ
**Core Principles**:
- ロックマンの青×白×黒カラーリングをバンドカラーに再解釈
- ピクセルアートの純粋な美しさを追求（アンチエイリアスなし）
- ゲームの「楽しさ」を最優先。情報アクセスがゲームプレイそのもの
- 夕暮れ～夜の都市ステージ（ロックマン2のワイリーステージ的雰囲気）

**Color Philosophy**: 夕暮れの空グラデーション（#1a0533→#4a1060→#c2185b）を背景に、プラットフォームは#1565c0（ロックマンブルー）、敵は#e53935（赤）、アイテムは#fdd835（黄）、テキストは#ffffff。

**Layout Paradigm**: 真横スクロール。地面から浮いたプラットフォーム群。ステージは左から右へ進行。各セクション（プロフィール・ディスコグラフィ・ライブ）がステージの「ボス部屋」として配置。

**Signature Elements**:

- ピクセルアートのキャラクタースプライト（青いロボット風）
- 星空の視差スクロール背景
- 情報ギミックは「宝箱」「ドア」「ボス」として表現

**Interaction Philosophy**: 矢印キー/WASDで移動、スペースでジャンプ、Zで攻撃。敵に触れると情報パネルが開く。

**Animation**: 60fpsのスムーズなキャラクター動作。ピクセルパーフェクトなスプライト切り替え。背景の星が流れる視差効果。

**Typography System**: Press Start 2P（見出し・UI）+ Noto Sans JP（日本語本文）
</text>

## 採用: アプローチ3「レトロ・ネオン・アーケード」

ロックマン直系のオマージュスタイルを採用。夕暮れ～夜の都市ステージ、青いロボットキャラクター、ピクセルアートの純粋な美しさを追求する。
